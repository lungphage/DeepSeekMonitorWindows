# 更新日志

本项目所有重要变更记录于此文件。版本号遵循语义化版本规范。

## [1.1.0] - 2026-06-08

### 新增

- 支持缓存命中、缓存未命中与输出 Token 的明细显示。
- 增加亮色 UI 皮肤，支持在主面板一键切换并记住用户选择。

## [1.0.1] - 2026-06-07

### 修复

- 修复应用单实例缺失导致的重复多开问题。此前在程序已运行的情况下，再次点击图标或 `exe` 会不断启动新的进程，且可以无限叠加（主窗口 `skipTaskbar` 不进任务栏，多开后不易察觉）。现接入 `tauri-plugin-single-instance` 单实例守卫：再次启动时不再新开窗口，而是把已有主面板显示并聚焦，第二个进程随即自行退出。

## [1.0.0] - 2026-06-07

### 新增

- 首个正式发布版本。
- DeepSeek API 账户余额查询，使用 DeepSeek 官方余额接口。
- DeepSeek 平台用量查询：当月消费、模型 Token 总量、请求数、缓存命中、缓存未命中、输出 Token。
- 支持 V4 Flash 与 V4 Pro 两类模型用量展示。
- 最近 7 天消费趋势图与模型详情页。
- Windows 托盘入口，主窗口默认不进入任务栏。
- API Key 保存、清除与余额验证。
- 用量 Token 网页登录自动同步与手动粘贴兜底。
- NSIS 安装包，安装程序与主程序均使用鲸鱼 logo 图标。
